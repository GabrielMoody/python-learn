String

Dalam istilah pemrograman, kita menyebut teks sebagai string. String merupakan kumpulan dari huruf huruf. Semua huruf, angka dan simbol yang ada di keyboard bisa menjadi sebuah String. Dalam hal ini, nama kalian bisa menjadi String dan juga alamat kalian.


Membuat String

Di Python, kita membuat string dengan menaruh tanda kutip diantara teks. Contoh, kita menggunakan variabel fred yang kita gunakan sebelumnya dan menggunakan variabel ini untuk menandai sebuah string.

Coba jalankan program Python berikut dengan menekan tombol dibawah

fred = "Python itu mudah dan menyenangkan" print(fred)
