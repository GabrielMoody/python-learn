List

"laba-laba, Ular, Singa, Katak" merupakan daftar nama-nama hewan dari sekian banyak hewan yang ada. Kita dapat menyimpan daftar nama hewan kedalam variabel kumpulan_hewan menggunakan string seperti ini.

kumpulan_hewan = "laba-laba, Ular, Singa, Katak" print(kumpulan_hewan)

Tetapi kita juga dapat membuat list. Sebuah jenis ajaib dari Python Object yang dapat kita manipulasi. Berikut contoh saat kita menggunakan list

kumpulan_hewan = ["laba-laba", "Ular", "Singa", "Katak"] print(kumpulan_hewan)
