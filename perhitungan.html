Perhitungan

Biasanya saat melakukan perhitungan antara 2 angka seperti 8 X 12, kamu akan menggunakan bantuan kalkulator atau menghitungnya secara manual dengan kertas dan pensil. Bagaimana jika perhitungan itu dilakukan menggunakan Python? Mari kita coba.

Coba jalankan program python berikut dengan menekan tombol dibawah

8 * 12

Saat melakukan perkalian menggunakan Python, kita tidak menggunakan simbol perkalian X melainkan simbol * (tanda bintang)

Kalian bisa melakukan operasi perkalian, penambahan, pengurangan dan pembagian menggunakan Python. Simbol yang digunakan Python untuk melakukan operasi matematika disebut operators, seperti tabel berikut

Simbol Operasi
+ Penambahan
- Pengurangan
* Perkalian
/ Pembagian
